# Urbanroof_Assignment
AI-powered DDR generator that processes inspection and thermal reports, extracts text and images, and creates structured, client-friendly diagnostic reports. Uses LLMs to merge insights, remove duplicates, and intelligently assign relevant images to report sections.
# DDR AI System

## What this does
This system reads inspection and thermal PDF reports, extracts text and images, and generates a structured DDR report using AI.

## Features
- Automatic text extraction
- Automatic image extraction
- Smart image selection
- DDR report generation

## How to run

1. Install dependencies:
pip install -r requirements.txt

2. Add your OpenAI API key in app.py

3. Run:
python app.py

## Output
- Extracted images folder
- Final DDR report in output folder
